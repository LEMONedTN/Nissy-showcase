## 📋 まとめ・使い方のヒント

| ツール | アスペクト比の指定方法 |
|---|---|
| **Midjourney** | プロンプト末尾に `--ar 3:2` のように追加 |
| **DALL-E 3** | 「1792×1024（横長）」「1024×1024（正方形）」「1024×1792（縦長）」から選択 |
| **Adobe Firefly** | キャンバスサイズで縦横比を設定 |
| **Stable Diffusion** | width / height パラメータで直接指定 |

### スタイル統一のために全プロンプトに追加する推奨タグ

```
photorealistic, professional photography, clean bright modern Japanese aesthetic, soft natural lighting, white and light blue color palette, no text, no watermark, high resolution
```
