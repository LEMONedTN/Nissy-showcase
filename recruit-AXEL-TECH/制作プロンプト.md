# 株式会社アクセルテック 採用コーポレートサイト 制作プロンプト

以下のプロンプトをそのままCoworkに貼り付けて実行すると、採用訴求のコーポレートサイト一式（HTML/CSS）が生成されます。
※フォルダは事前に用意してから実行してください。画像はあとから差し替える前提で、画像箇所は `#ccc` のプレースホルダーにしてあります。

---

## 基本情報
- 業種：IT・システム開発／ITソリューション（SES・受託開発）
- 屋号：株式会社アクセルテック（AXEL TECH Inc.）
- 代表者名：代表取締役 山田 健一
- ターゲット：20〜40代のビジネスパーソン（主にエンジニア・営業人材）

## 参考サイト
- テイスト参考：採用サイト｜株式会社コサウェル（https://cosawell-recruit.jp/）
  → 落ち着いた配色、シンプルでスタイリッシュなフラットデザイン。余白を広くとり、信頼感・先進性・専門性が伝わるトーン。ネイビー基調＋ブルーアクセント。
- レイアウト参考：おまかせ（採用サイトの王道構成で設計）
  → フルスクリーンのファーストビュー → 数字で見る会社 → 各コンテンツへの導線カード → CTA（エントリー）という縦スクロール型。

## デザイン方針（トーン＆雰囲気）
- 落ち着いた配色／シンプル／スタイリッシュ／フラットデザイン
- 信頼感・先進性・専門性が伝わること
- 装飾は最小限。余白・タイポグラフィ・整列で品よく見せる
- 影は薄く（フラット寄り）、角丸は控えめ（4〜8px）

## カラーパレット（CSS変数で定義）
- ベース（背景）：#FFFFFF
- プライマリ（ネイビー）：#102A47
- アクセント（ブルー）：#2E80E6
- サブ（淡グレー背景）：#F5F7FA
- ボーダー：#E2E8F0
- テキスト：#1A2733
- サブテキスト：#5C6B7A

## フォント
- 日本語：Noto Sans JP（Google Fonts）
- 英字・見出し数字：Montserrat（Google Fonts）

## 出力ファイル構成
```
index.html        … 採用TOP
about.html        … 会社説明
interview.html    … 社員インタビュー（個別ページは作らず、モーダル/ポップアップで表示）
roundtable.html   … 座談会
workflow.html     … 1日の仕事の流れ（1部署分）
workstyle.html    … 働き方／福利厚生
recruit.html      … 採用情報／採用フォーム
style/
├── reset.css     … テンプレ固定（ブラウザ独自CSSの解除）
└── style.css     … デザイン適用
```

## ナビゲーション（全ページ共通ヘッダー）
採用TOP / 会社説明 / 社員インタビュー / 座談会 / 1日の仕事の流れ / 働き方・福利厚生 / 採用情報
- ロゴ（左）＋ナビ（右）。スマホはハンバーガーメニュー。
- 現在のページはアクティブ表示（色・下線）。
- ヘッダー右端に「エントリー」ボタン（アクセントカラー、recruit.html#formへ）。

## 各ページの構成要件

### index.html（採用TOP）
- ファーストビュー：キャッチコピー＋サブコピー＋エントリーボタン＋背景画像プレースホルダー（#ccc）。
  キャッチ例：「技術で、未来をアクセラレートする。」
- 数字で見る株式会社アクセルテック：平均年齢・エンジニア比率・有給取得率・リモート可率などを大きな数字で4〜6項目。
- 私たちについて（会社説明への導線、画像＋テキスト2カラム）。
- コンテンツ一覧カード：社員インタビュー／座談会／1日の流れ／働き方・福利厚生 へのリンクカード（画像プレースホルダー＋見出し）。
- 募集職種ハイライト：エンジニア・営業の2職種を簡潔に。
- 最下部CTA：「あなたのキャリアを、ここから。」＋エントリーボタン。

### about.html（会社説明）
- ページタイトル＋パンくず。
- 代表メッセージ（顔写真プレースホルダー＋メッセージ、署名：代表取締役 山田 健一）。
- ミッション／ビジョン／バリュー（3カラム）。
- 事業内容（受託開発・SES・自社サービスなど3項目、アイコンは#ccc四角で代替可）。
- 会社概要テーブル（社名・設立・資本金・代表者・所在地・従業員数・事業内容。住所/電話はダミー可）。

### interview.html（社員インタビュー）
- 社員カードをグリッド表示（6名程度。顔写真プレースホルダー＋氏名＋職種＋入社年）。
- カードクリックでモーダル（ポップアップ）が開き、Q&A形式の詳細インタビューを表示。
  → 個別ページは作らない。JavaScriptでモーダル開閉（背景クリック・×ボタン・Escで閉じる）。
- 質問例：入社の決め手／仕事のやりがい／一日の過ごし方／今後の目標。
- エンジニア・営業の両職種をバランスよく。

### roundtable.html（座談会）
- テーマ設定（例：「若手×中堅で語る、アクセルテックのリアル」）。
- 参加者紹介（3〜4名、顔写真プレースホルダー＋ニックネーム）。
- 対話形式（発言者アイコン＋吹き出し or 名前ラベル＋発言）でトーク展開を複数トピック。
- 締めのメッセージ＋エントリー導線。

### workflow.html（1日の仕事の流れ・開発部門）
- 対象部署を明記（例：開発部 エンジニア）。
- タイムライン形式（9:00〜19:00など）で各時刻＋見出し＋説明＋画像プレースホルダー。
- 縦のタイムライン（左に時刻、右に内容）。スマホでも崩れない設計。
- 末尾に「先輩社員の声」を一言。

### workstyle.html（働き方／福利厚生）
- 働き方の特徴（リモート・フレックス・副業可など）をカードで。
- 福利厚生一覧（住宅手当・資格取得支援・書籍購入補助・健康診断など）をグリッドで。
- 研修・キャリア支援（オンボーディング、メンター、スキルアップ）。
- 数字や制度名は具体的に（ダミーで可）。

### recruit.html（採用情報／採用フォーム）
- 募集要項テーブル（職種・雇用形態・給与・勤務地・勤務時間・休日休暇・福利厚生）をエンジニア／営業の2職種分。
- 選考フロー（エントリー→書類→面接→内定の4ステップ、横並びステップUI）。
- 採用フォーム（id="form"）：氏名・フリガナ・メール・電話・希望職種（select）・自己PR（textarea）・個人情報同意（checkbox）・送信ボタン。
  → action未設定でも可。必須項目はrequired。送信時はJSでサンクスメッセージ表示でも可。

## 共通仕様
- 全ページ共通のヘッダー・フッターを統一すること。
- ナビゲーションは現在ページをアクティブ表示。
- フッター：会社名・ナビ簡易リンク・SNSダミーリンク・プライバシーポリシー（テキストリンクのみ、別ページ不要）・コピーライト。
- Google Fonts使用（Noto Sans JP / Montserrat）。
- `<meta charset="UTF-8">` と viewport を必ず含める。
- レスポンシブ対応必須（スマホ：768px以下でレイアウト最適化、ハンバーガーメニュー）。
- 画像が入る箇所はすべて `background-color: #ccc`（または #ccc 背景のdiv）のプレースホルダーで代替。imgは使わずプレースホルダーdiv推奨。
- ダミーテキストはIT企業・エンジニア／営業ターゲットに合った自然な日本語にする（「テキストが入ります」は使わない）。
- 住所・電話番号などはダミーで問題なし。
- コピーライト：© 2025 株式会社アクセルテック All Rights Reserved.
- HTML/CSSのみ＋必要最小限のVanilla JS（モーダル・ハンバーガー・フォーム）。外部ライブラリ不要。
